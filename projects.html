Ray Cast maze game

This is a game I made in python with pycharm. I used a method called raycasting to achieve the 3D effect. Ray casting is essentially sending out a bunch of rays (lines) out from the player's POV. The ray stops when it hits a wall, calculates the distance and draws a line vertically on the screen proportional to how big the ray is. The goal of the game is to make your way through a bunch of mazes as fast as you can.

8-bit computer

This is the architecture of an 8-bit computer I built from scratch. Meaning that I put together all the gates together to make one turing complete machine. While this was done in a game called turing complete, I had no help in coming up with the ways to connect the gates. The game simply guided me along to build every component of the computer I would need for it to be turing complete. What I mean by turing complete is that it can solve any computable problem given enough time and memory. This computer can only be coded in assembly and the ISA is specific to the game.
